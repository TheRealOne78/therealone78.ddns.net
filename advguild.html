About

This webpage serves as a HTTP(S) mirror for the advguild.myftp.org ftp server.
This domain also provides FTP service for the mirror, running at ftp://therealone78.ddns.net/html/advguildRoot. To access FTP, use a FTP client like the default terminal FTP client or FileZilla with username anonymous and password anonymous.

Port 22 SSH/SFTP is open, but can't be accessed by anonymous people, so SFTP mirror service is not available for this domain. FTPS with port 990 is closed, so FTPS mirror service is not available for this domain.

The official server is advguild.myftp.org, located in Canada.
