About

This webpage serves as a http(s) mirror for advguild.myftp.org ftp server.
This DNS provides as well ftp service for the mirror, running at ftp://therealone78.ddns.net/advguild. To access FTP, please use a FTP client like the default terminal FTP client or FileZilla with username anonymous and no password.
The official server is advguild.myftp.org, located in Canada.
