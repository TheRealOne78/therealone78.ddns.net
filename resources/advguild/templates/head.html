About

This webpage mirrors the advguild.myftp.org FTP server.

You can also access the mirror via FTP or FTPS on port 21 at ftp://therealone78.ddns.net/advguildRoot and ftps://therealone78.ddns.net/advguildRoot. Use an FTP client like the terminal FTP tool or FileZilla with username anonymous and password anonymous. Port 990 is closed, so FTPS connections use port 21.

SFTP access to this mirror is not yet available despite port 22 (SSH/SFTP) being currently open.

The official server is advguild.myftp.org, located in Canada.
